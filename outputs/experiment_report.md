# 实验报告

## 数据结构

- 教师数据：`I/J/K/C/D` 五个 sheet，分别对应工厂、候选仓库、客户、一级运输成本、二级运输成本。
- 论文输入要求：多产品、多层容量、坐标、体积容量。
- 教师数据适配方式：保留原始单产品流与成本矩阵，补齐 3 层产品容量、3 层体积容量、随机坐标与单位产品体积。

## 代码结构

- `src/scflp_multilevel/exact_models.py`：Model A / Model B 的 Gurobi 精确模型。
- `src/scflp_multilevel/maat.py`：MAAT 三阶段流程、局部搜索、MPTP 子问题。
- `src/scflp_multilevel/teacher_adapter.py`：教师数据适配。
- `run_reproduction.py`：单实验入口。
- `run_batch_resume.py`：批量实验、断点续跑、汇总表刷新。
