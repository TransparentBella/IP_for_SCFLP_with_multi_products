# 复现核对清单

## 严格按论文复现的部分

- 两个精确模型都已实现：
  - Model A: 公式 (1) 到 (10)
  - Model B: 公式 (11) 到 (19)
- `rho` 的计算已按论文式 (20) / (21) 实现。
- `mu = beta * rho` 已实现。
- MAAT 三阶段结构已实现：
  - Stage 1: 聚合候选仓库 + 局部搜索
  - Stage 2: 原问题局部搜索
  - Stage 3: 在 `S*` 上做精确求解
- 局部搜索已按 Fig. 2 实现：
  - 使用 first-improvement
  - 候选仓库先按最近 opened depot 归类
  - 计算每个 opened depot 的 `d_hat_s`
  - 仅当 `d_js < gamma * d_hat_s` 时尝试交换
  - 每次交换后重新最优求解 MPTP
- MPTP 子问题已按论文处理为连续流 LP。
- 局部搜索中所有 opened depot 使用最大容量，已按论文处理。
- 默认 MAAT 参数已按论文第 4 节设置：
  - `T = 10`
  - `beta = 1.5`
  - `tau = 150`
  - `epsilon = 0.5%`
  - `tau' = 108`
  - `tau_hat = 0.25 * n * |P|`
  - `gamma = 2.5`
- Model B 的 `beta` 特例也已实现：
  - `|P| = 10 -> beta = 2`
  - `|P| = 5 -> beta = 3`

## 非严格逐表复刻的部分

- 论文没有完整公开随机数据生成全部公式。
- 老师数据不是论文原始输入结构：
  - 只有单产品
  - 只有单层容量
  - 没有坐标
  - 没有产品体积

因此以下部分是显式建模选择，不应宣称为“论文原文已给出”：

- 随机数据生成器中的坐标采样区间
- 固定成本标定公式
- Model B 的产品体积生成规则
- 老师数据到多层容量实例的适配规则
- 老师数据适配时补齐的坐标

## 当前结论

- 如果要求“严格复现论文模型与算法”，当前实现满足要求。
- 如果要求“严格重建论文原始随机数据表”，当前实现不满足，因为原文未完整披露生成公式。
- 如果要求“直接用老师 Excel 严格跑论文 MAAT”，原始数据本身不满足论文输入结构，所以只能做兼容适配，不能称为完全同构复现。
